## Overview

This API enables bulk validation of Udyog Aadhaar numbers through an asynchronous verification process. You submit a batch file containing Udyog Aadhaar numbers, and the system processes these requests in the background. Results are delivered via webhook callback to your registered endpoint.

<Callout kind="alert">
  Input files must contain fewer than 1,000 records and use `udyog_aadhaar_number` as the sole column header.
</Callout>
