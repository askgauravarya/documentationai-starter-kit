## Overview

The Mobile Number Intelligence APIs enable you to find, verify, and fetch multiple data points associated with a mobile number, including:

- Full Name
- KYC Details
- UPI ID
- IFSC Code
- Network Details
- Digital Age
- Employment Information (UAN)
- Vehicle RC

## Available APIs

| API | Description |
|-----|-------------|
| [Mobile To Name](/reference/mobile-number-intelligence/mobile-to-name) | Get the name associated with a mobile number |
| [Mobile To PAN](/reference/mobile-number-intelligence/mobile-to-pan) | Get the PAN linked to a mobile number |
| [Mobile To UAN](/reference/mobile-number-intelligence/mobile-to-uan) | Get the UAN linked to a mobile number |
| [Mobile To UAN List](/reference/mobile-number-intelligence/mobile-to-uan-list) | Get all UANs linked to a mobile number |
| [Mobile To UPI](/reference/mobile-number-intelligence/mobile-to-upi) | Get the UPI ID linked to a mobile number |
| [Mobile To Multiple UPI](/reference/mobile-number-intelligence/mobile-to-multiple-upi) | Get all UPI IDs linked to a mobile number |
| [Mobile To Vehicle RC](/reference/mobile-number-intelligence/mobile-to-vehicle-rc) | Get vehicle RC details linked to a mobile number |
| [Mobile To Digital Age](/reference/mobile-number-intelligence/mobile-to-digital-age) | Get the digital age of a mobile number |
| [Mobile To Network Details](/reference/mobile-number-intelligence/mobile-network-details) | Get network and operator details for a mobile number |
| [MNRL Check](/reference/mobile-number-intelligence/mnrl-check) | Check if a mobile number is in the MNRL (Mobile Number Revocation List) |
