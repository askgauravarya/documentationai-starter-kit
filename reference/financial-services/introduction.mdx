## Overview

The Financial Services API suite provides a whole array of APIs to serve multiple financial services and fetch data for an organization or individual from various data sources in a streamlined manner.

## Key capabilities

1. **Credit Bureau Reports** — Fetch credit reports from partner credit bureaus like Equifax
2. **Bank Statement Analysis** — Upload and analyze bank statements for automated financial insights
