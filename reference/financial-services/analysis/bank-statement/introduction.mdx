## Overview

The Bank Statement Analysis API transforms bank statements into actionable financial intelligence through automated extraction and analysis.

## Key capabilities

- **Fast extraction** — Upload and process bank statements within minutes
- **Smart categorization** — Auto-categorizes income, expenses, EMIs, utilities, and more
- **Financial summaries** — Get total credits, debits, average balances, and spending patterns
- **Secure processing** — End-to-end encryption ensures your data stays protected

## Integration workflow

<Steps>
  <Step title="Upload statement" icon="upload">
    Upload a bank statement file using the [Upload Statement](/reference/financial-services/analysis/bank-statement/upload) API endpoint.
  </Step>

  <Step title="Download report" icon="download">
    Retrieve the analyzed report with categorized transactions and financial summaries using the [Download Report](/reference/financial-services/analysis/bank-statement/download-report) API endpoint.
  </Step>
</Steps>
