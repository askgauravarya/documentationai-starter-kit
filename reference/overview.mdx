## Overview

Deepvue provides 150+ REST APIs across identity verification (KYC), business verification (KYB), document OCR, financial services, and more. Whether you are onboarding individuals, verifying businesses, or running background checks, Deepvue offers a unified platform to streamline your compliance workflows.

**Base URL:**

```
https://production.deepvue.tech
```

All API requests require authentication via a bearer token and your API key. Generate a token using the `/v1/authorize` endpoint before making any other API calls.
