## Prerequisites

<Callout kind="info">
  You will need:
  - A Deepvue account with API credentials from the [Dashboard](https://dashboard.deepvue.tech)
  - cURL, Postman, or your preferred programming language
</Callout>

**Base URL:**

```
https://production.deepvue.tech
```

## Make your first API call

<Steps>
  <Step title="Get your credentials" icon="key">
    Log in to the [Deepvue Dashboard](https://dashboard.deepvue.tech) and navigate to the credentials tab. Copy your `client_id` and `client_secret`.
  </Step>

  <Step title="Generate an access token" icon="lock">
    Call the Authorize endpoint to get a Bearer token.

    ```bash
    curl -X POST 'https://production.deepvue.tech/v1/authorize' \
      -F 'client_id=YOUR_CLIENT_ID' \
      -F 'client_secret=YOUR_CLIENT_SECRET'
    ```

    You will receive a response like:

    ```json
    {
      "access_token": "eyJhbGciOiJIUzI1NiIs...",
      "token_type": "bearer",
      "expiry": "2026-03-08T00:00:00Z"
    }
    ```

    <Callout kind="tip">
      Access tokens are valid for 24 hours. Store the token securely for use in subsequent requests.
    </Callout>
  </Step>

  <Step title="Verify a PAN number" icon="play">
    Use the [PAN Basic](/reference/kyc/pan/pan-verification-basic) API to verify a PAN number.

    <CodeGroup tabs="cURL,Python,Node.js">
      ```bash
      curl -X GET 'https://production.deepvue.tech/v1/verification/panbasic?pan_number=AAAPT0002F' \
        -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
        -H 'x-api-key: YOUR_CLIENT_SECRET'
      ```

      ```python
      import requests

      headers = {
          'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
          'x-api-key': 'YOUR_CLIENT_SECRET'
      }
      response = requests.get(
          'https://production.deepvue.tech/v1/verification/panbasic',
          params={'pan_number': 'AAAPT0002F'},
          headers=headers
      )
      print(response.json())
      ```

      ```javascript
      const response = await fetch(
        'https://production.deepvue.tech/v1/verification/panbasic?pan_number=AAAPT0002F',
        {
          headers: {
            'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
            'x-api-key': 'YOUR_CLIENT_SECRET'
          }
        }
      );
      const data = await response.json();
      console.log(data);
      ```
    </CodeGroup>
  </Step>

  <Step title="Check the response" icon="check-circle">
    A successful response looks like:

    ```json
    {
      "code": 200,
      "timestamp": 1738040933033,
      "transaction_id": "ece94d5e2749482faa10184962e9e34a",
      "data": {
        "@entity": "pan",
        "pan": "AAAPT0002F",
        "full_name": "RATAN NAVAL TATA",
        "status": "VALID",
        "category": "Individual",
        "name_information": {
          "pan_name_cleaned": "Ratan Naval Tata"
        }
      }
    }
    ```
  </Step>
</Steps>
