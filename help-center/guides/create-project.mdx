## Overview

Set up a new project in deepvue in minutes.

## Steps

<Steps>
  <Step title="Navigate to Dashboard">
    Go to `https://deepvue.io/dashboard` and log in.
  </Step>
  <Step title="Click New Project">
    Select the "New Project" button at the top.
  </Step>
  <Step title="Enter Details">
    Add project name and select a documentation template.
  </Step>
  <Step title="Finish">
    Click "Create" to launch your project space!
  </Step>
</Steps>
