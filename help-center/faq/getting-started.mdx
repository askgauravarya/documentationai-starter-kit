## Overview

To use deepvue, create an account and set up your first project.

## Steps

1. Go to `https://deepvue.io/signup`
2. Register with your email
3. Log in and click "New Project"
4. Name your project and choose a template
5. Click "Create" to get started

That's it! You can now organize documentation using deepvue's dashboard.
