## Platform essentials

| Feature | Details |
|---------|---------|
| **Base URL** | `https://production.deepvue.tech` |
| **Protocol** | HTTPS required, REST conventions |
| **Auth** | Bearer token + API key ([details](/authentication)) |
| **Response format** | JSON with `code`, `timestamp`, `transaction_id`, `data` |
| **Rate limits** | Plan-based — 1 to 17 RPS ([details](/rate-limits)) |
| **Error handling** | Standard HTTP codes with `detail` field ([details](/errors)) |
