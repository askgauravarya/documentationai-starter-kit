## Overview

Deepvue provides API infrastructure for financial integrations and insights. You can launch financial products faster with APIs covering identity verification, banking, credit scoring, and more.

The Deepvue APIs follow REST conventions with resource-oriented URLs, support for query parameters and form-encoded requests, JSON responses, and standard HTTP status codes and authentication methods.

**Base URL:**

```
https://production.deepvue.tech
```

## Get started

<Steps>
  <Step title="Access the API Dashboard" icon="layout-dashboard">
    Sign up or log in to the [Deepvue Dashboard](https://dashboard.deepvue.tech) to get your API credentials. No code required to get started.
  </Step>

  <Step title="Get your API keys" icon="key">
    Navigate to the credentials section in your dashboard. Trial account keys have the prefix **free_tier** and live account keys have the **live** prefix followed by your organization ID.
  </Step>

  <Step title="Authenticate" icon="lock">
    Use your `client_id` and `client_secret` to generate an access token via the [Authorization](/reference/auth/authorization) endpoint. Include the token in subsequent API calls.
  </Step>

  <Step title="Make your first API call" icon="play">
    Try a simple verification like [PAN Basic](/reference/kyc/pan/pan-verification-basic) to confirm your integration is working.
  </Step>
</Steps>

<Callout kind="tip">
  Use our [Postman collection](https://www.postman.com) for ready-to-use requests, or fetch the complete documentation index at [docs.deepvue.ai/llms.txt](https://docs.deepvue.ai/llms.txt).
</Callout>

## Supported languages

Deepvue API supports integration with any language that can make HTTP requests. Code examples are available in:

- cURL
- Ruby
- Python
- PHP
- Java
- Node.js
- Go
